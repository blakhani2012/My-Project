Example of while Loop

package basic;

public class JavawhileLoop {
	
	public static void main(String arg[])
	
	{
		int a=12;
		
		 while(a < 10)
		{
			 System.out.println(a);
			 a++;
			
		}
		 
		 int b=12;
		 
		 do
		 {
			 System.out.println("value of b is "+ b);
			 b++;
		 }
		 while(b < 10);
		
	}

}

output:

value of b is 12

Difference between while loop and do while

-->while loop
   --it will execute until condition is true.
   
-->do while loop
   --it will first execute  after that it will check condition.
     so it will execute at least once.
