Control Statement

java provide following control statement

if block

else

else if

In java if is condition block.
let's see some example.

If.java

package Controlstatement;

public class If 
{

	public static void main(String[] args) 
	{
	    int a=10;
		
		if(a>0)
		{
			System.out.println("a is greater zero");
			
		}
			
	}

}

else if 

package Controlstatement;

public class If 
{

	public static void main(String[] args) 
	{
	    String name="bhavesh";
	    
		if(name=="bhavesh")
		{
			System.out.println(" Hello bhavesh");
			
		}
		
		else if (name=="vivek") 
		{
		
		 System.out.println("hello vivek");
		}	
	}
}

Output: Hello bhavesh
