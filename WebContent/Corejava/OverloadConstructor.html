Constructor overloading


--> in a class we can define any number of constructor is know as overloaded 
    constructor.

--> whenever we create overloaded constructor argument should be diffrr in
    terms argument type or argument length.

--> the overloaded constructor are used to create object with different 
    initialization

--> constructor returns the object address which is constructed by constructor
    of class.

Constrcutor overloading

package constructor;

public class OverLoadconstructor 
{

	int a;
	
	
	public OverLoadconstructor() 
	{
		System.out.println(a);
	}
	
	OverLoadconstructor(int a)
	{
		this();
		this.a=a;
		System.out.println(a);
	}
}


class Mainclass
{
	public static void main(String[] args) 
	
	{
		OverLoadconstructor a = new OverLoadconstructor(28);
	}
}

28
