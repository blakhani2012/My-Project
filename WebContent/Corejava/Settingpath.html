How to set path 

You can set path in two ways.
  
   1.temporary
   2.permanent

1.Temporary 

Step to set path for temporary

1.Open cmd

2.copy the path of jdk/bin directory
   Exa. in My case copied path  C:\Program Files (x86)\Java\jre1.8.0_25\bin
   
3.Type command in cmd
     set path=C:\Program Files (x86)\Java\jre1.8.0_25\bin

2.Permanent 

Step to set path permenent

1.right click on my computer and open Properties
2.click on advanced system setting
3.after that click on environment variable
4.Under System Variable Find option path
  and copied your jdk/bin path
   
   in my case Copied path  is:
   
   C:\Program Files (x86)\Java\jre1.8.0_25\bin;


 Note: put Semi colon at the last
